## Tic tac toe game

Key requirements:
- python 3.5
- aiohttp;
- SQLAlchemy;
- aiomysql;
- jinja2;
- aiohttp_jinja2;
- aiohttp_session;
- aiohttp_debugtoolbar;

Short sample *.env* file with settings.

```
DEBUG='True'
MYSQL_HOST='host'
MYSQL_DB_NAME='dbname'
MYSQL_USER='user'
MYSQL_PASSWORD='password'
SECRET_KEY='secret_key'
HOST='127.0.0.1'
PORT=8000
```
